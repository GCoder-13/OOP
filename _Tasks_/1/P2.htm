Для початку коротка інформація: TDateTime - по суті всього лише число з плаваючою точкою подвійної точності, з усіма витікаючими наслідками. Тобто, наприклад, для того, щоб обчислити різницю між двома датами, достатньо відняти одне з іншого:

TDateTime datetime1 = ...;

TDateTime datetime2 = ...;

double difference = (double) (datetime2 - datetime1);

При цьому вийшло значення представлятиме точне значення з урахуванням днів, годин, хвилин і секунд. Наприклад, значення 3.25 - це три дні і ще 0.25 дня (тобто 6:00)

Як дізнатися поточні дату і / або час?

Немає нічого простіше:

TDateTime CurrentDate = Date (); // Це поточна дата

TDateTime CurrentTime = Time (); // Це поточний час

TDateTime CurrentDateTime = Now (); // Це поточні дата і час

Як зрушити дату на заданий проміжок?

TDateTime datetime = ...;

datetime + = 1.0 / 24; // Зрушуємо на годину вперед

datetime - = 1.0; // Зрушень на день назад

datetime + = 7.0; // Зрушуємо на тиждень вперед

datetime = IncMonth (datetime -1); // Зрушуємо на місяць назад

Як визначити день тижня?

TDateTime datetime = ...;

int day_of_week = (datetime.DayOfWeek () + 5)% 7; // Вважаємо з понеділка і з нуля

Як визначити номер тижня у році?

Слід мати на увазі, що перший тиждень в році може бути неповною.

int week_num;

Word year, month, day;

TDateTime dt = ...; // "Піддослідна" дата

dt.DecodeDate (& year, & month, & day); // Розкладаємо нашу дату на складові

TDateTime _1_jan (year, 1,1); // 1-е січня того року, якому відповідає дата

int _1_jan_day_of_week = (_1_jan.DayOfWeek () + 5)% 7; // День тижня 1-го січня

week_num = (int) (dt-TDateTime (year, 1,1) + _ 1_jan_day_of_week) / 7; // Шуканий номер тижня (з 0)

Як дізнатися, скільки днів у місяці?

Кількість днів у місяці зберігається в глобальному масиві MonthDays, отримати значення можна так:

MonthDays [IsLeapYear (номер_года)] [номер_месяца]

Як змінити один з компонентів TDateTime, не зачіпаючи інший?

TDateTime datetime1 = ...;

TDateTime datetime2 = ...;

ReplaceDate (datetime1, datetime2); // Datetime1 тепер містить ту ж дату, що і datetime2

// Але час залишилося колишнім

ReplaceTime (datetime2, datetime1); // А тепер datetime2 містить той же час, що і datetime1

// Але дата залишилася колишньою

Як визначити, чи є рік високосним?

Для цього треба використовувати функцію IsLeapYear. Наприклад, наступний код покаже вікно з текстом про те, чи є рік високосним чи ні:

int year = 2100; // Тут какой треба рік

ShowMessage (IntToStr (year) + "рік -" + ((IsLeapYear (year))? "": "Не") + "високосний");

Як видно, функція IsLeapYear приймає параметр - рік і повертає true, якщо він високосний і false в іншому випадку.
